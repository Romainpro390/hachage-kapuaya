# Algorithme de Hachage Personnalisé - KAPUAYA KAJA

## Description
Ce projet présente une fonction de hachage unique développée en Python. L'objectif est de transformer une chaîne de caractères en une empreinte hexadécimale en utilisant une logique mathématique basée sur les valeurs ASCII et une signature personnelle.

## Caractéristiques
- **Nom de la fonction :** `hash_kapuaya_kaja`
- **Signature (Salt) :** L'algorithme utilise la chaîne "KAPUAYA_KAJA" pour garantir l'unicité du hachage.
- **Logique :** Multiplication par des nombres premiers et pondération par la position des caractères.
